## 🚀 方案1：Streamlit Cloud（强烈推荐）

### 优势
- ✅ 官方免费托管
- ✅ 一键部署
- ✅ 自动更新
- ✅ 无需配置
- ✅ 支持手机访问

### 部署步骤

#### 第1步：准备文件
确保你有以下文件：
```
streamlit_app/
├── app.py                    # 主应用
├── requirements.txt          # 依赖包
└── .streamlit/
    └── config.toml          # 配置文件
```

#### 第2步：上传到GitHub
1. 创建一个GitHub仓库
2. 上传上述文件到仓库
3. 确保文件在仓库根目录

#### 第3步：部署到Streamlit Cloud
1. 访问：https://share.streamlit.io/
2. 点击「Deploy now」
3. 使用GitHub账号登录
4. 选择你的仓库
5. 选择 `app.py` 作为入口文件
6. 点击「Deploy」

#### 第4步：等待部署
- 部署通常需要1-2分钟
- 部署成功后会获得一个链接

#### 第5步：手机访问
- 在手机浏览器打开部署链接
- 开始使用！

### 示例链接格式
```
https://your-app-name.streamlit.app/
```

## 🌐 方案2：部署到Vercel（备选）

### 注意
Vercel是Serverless平台，部署Streamlit需要特殊配置。

### 部署步骤

#### 第1步：创建启动脚本
创建一个 `index.py` 文件：

```python
import os
import subprocess

# 设置环境变量
os.environ['STREAMLIT_SERVER_PORT'] = '8501'
os.environ['STREAMLIT_SERVER_ADDRESS'] = '0.0.0.0'

# 启动Streamlit
subprocess.run(['streamlit', 'run', 'app.py'])
```

#### 第2步：修改 vercel.json
```json
{
  "version": 2,
  "builds": [
    {
      "src": "index.py",
      "use": "@vercel/python"
    }
  ]
}
```

#### 第3步：上传到GitHub
上传所有文件到GitHub仓库

#### 第4步：连接Vercel
1. 访问：https://vercel.com/
2. 点击「New Project」
3. 导入GitHub仓库
4. 点击「Deploy」

#### 第5步：配置环境变量（可选）
在Vercel中添加：
```
STREAMLIT_SERVER_PORT = 8501
STREAMLIT_SERVER_ADDRESS = 0.0.0.0
```

## 📱 手机使用指南

### 访问方式
1. 在手机浏览器打开应用链接
2. 点击「上传视频」按钮
3. 选择视频文件
4. 点击「生成文案」
5. 查看并复制文案

### 浏览器兼容性
- ✅ Chrome（推荐）
- ✅ Safari
- ✅ Edge
- ✅ 微信内置浏览器
- ✅ 其他现代浏览器

### 文件要求
- **格式**：MP4, MOV, AVI, WMV, MKV
- **大小**：建议小于500MB
- **时长**：建议1-10分钟
- **画质**：清晰度越高效果越好

## 🔧 配置自定义

编辑 `app.py`，找到：

st.set_page_config(

## 🐛 常见问题

### Q1: 部署失败怎么办？
**解决**：
- 检查 `requirements.txt` 是否存在
- 确认 `app.py` 文件名正确
- 查看部署日志

### Q2: 视频无法上传？
**解决**：
- 检查视频格式是否支持
- 确认视频大小是否超限
- 尝试使用Chrome浏览器

### Q3: 文案生成失败？
**解决**：
- 检查网络连接
- 重新上传视频
- 尝试较小的视频文件

### Q4: 手机访问很慢？
**解决**：
- 使用WiFi网络
- 检查视频大小
- 更换浏览器

### Q5: 如何更新应用？
**解决**：
- **Streamlit Cloud**：自动更新（修改代码后等待1-2分钟）
- **Vercel**：重新部署或在GitHub提交新代码
